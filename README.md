# Project Setup 

### Step-1
- Make sure you have installed python 3.11
- Create virtual environment by using following commands
> $ pip install virtualenv
> 
> $ python -m venv myenv
> 
> $ myenv\Scripts\activate.bat
> 
### Step-2
Install the dependencies
> $ pip install -r requiremensts.txt
### Step-3
## For Dev Environment
Apply migrations
> $ py manage.py makemigrations
> 
> $ py manage.py migrate
### Step-4
Run the project 
> $ py manage.py runserver
